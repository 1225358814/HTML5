1、创建html文件（文件后缀名是.html）
2、通过VSCode打开第一步创建的html文件
3、开始编写基本的html文件结构

<html>
    <head>
        <meta charset="utf-8">
        <title>网页的标题</title>
    </head>
    <body>
        
    </body>
</html>

4、根据需要编写对应标签

5、基本的样式选择器结构
            /*
            将要设置样式的那个标签{
                将要设置的属性：属性值；
            }
            */
            body{
                /*字体颜色*/
                color: green;
                /*背景颜色*/
                background-color: black;
            }
            p{
                /*字体*/
                font-family: "楷体";
                color: lightseagreen;
                /*字体大小：px是像素pixel的缩写*/
                font-size: 30px;
                /*文字对齐方式：left、center、right*/
                text-align: right;
            }

备注：
复制、粘贴、剪切：常规的ctrl + C 、V 、X
快速对齐的快捷键： alt + shift + F
VSCode关闭/打开左侧菜单栏：ctrl + B
快速注释选中代码：ctrl + /
